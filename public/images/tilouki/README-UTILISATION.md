# Pack photos Tilouki 2026

Ce dossier contient 52 photos classees par usage pour eviter les visuels aleatoires.

## Structure

- 01-categories : visuels de categories principales.
- 02-rituels-et-moments : modules emotionnels comme Nuit calme, Jour de pluie, Lecture du soir.
- 03-home-et-marque : images fortes pour page d'accueil, sections de marque et selections.
- 04-blog : couvertures d'articles.
- 05-details-et-matieres : details textile, flat lays et matieres.
- 06-reassurance-et-guides : guides, tailles, confort, messages de confiance.

## Utilisation conseillee

1. Ouvrir galerie-verification.html pour verifier rapidement toutes les images.
2. Lire manifest-photos-tilouki.csv pour savoir quel fichier correspond a quel module.
3. Renommer les chemins dans le site en gardant les noms de fichiers explicites.
4. Pour les visuels avec enfants identifiables, ne pas laisser entendre que l'enfant ou le photographe recommande Tilouki.

## Important

Les photos viennent principalement d'Unsplash, avec un visuel Pexels pour l'entretien du linge bebe. L'attribution n'est pas obligatoire selon ces licences, mais elle est conservee dans CREDITS-LICENCES.md par transparence.
